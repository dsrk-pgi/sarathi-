# Assets Directory

This directory contains all the static assets for the Sarathi app.

## Required Assets

To complete the app setup, please add the following image files:

1. **icon.png** (1024x1024) - App icon
2. **splash.png** (1242x2436) - Splash screen image
3. **adaptive-icon.png** (1024x1024) - Android adaptive icon
4. **favicon.png** (48x48) - Web favicon
5. **notification-icon.png** (96x96) - Notification icon

## Design Guidelines

- **Colors**: Use Deep Royal Blue (#1E3A8A) and Golden Yellow (#F59E0B)
- **Logo**: Should feature the "सारथी" text with a navigation/guidance theme
- **Style**: Modern, clean, and trustworthy

## Temporary Placeholders

For development, you can use solid color placeholders:
- Background: Deep Royal Blue (#1E3A8A)
- Accent: Golden Yellow (#F59E0B)

You can generate these using online tools or design software like Figma, Canva, or Adobe Illustrator.
